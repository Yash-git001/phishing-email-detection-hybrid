# Phishing Email Detection System (Hybrid Approach)

## 📌 Overview
This project is a phishing email detection system that combines:
- Rule-based detection
- Machine learning (Logistic Regression with TF-IDF)

It classifies emails as:
- Legitimate (0)
- Phishing (1)

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Streamlit

---

## 🧠 How It Works
1. Input email text
2. Preprocessing (cleaning, tokenization)
3. Feature extraction using TF-IDF
4. Machine learning prediction
5. Rule-based scoring
6. Hybrid decision output

---

## 🚀 How to Run the Project

### Step 1: Install dependencies

pip install -r requirements.txt

### Step 2: Run the app

python main.py

streamlit run app.py



---
## 📊 Results
- ML Accuracy: ~96%
- Hybrid Accuracy: ~94%
